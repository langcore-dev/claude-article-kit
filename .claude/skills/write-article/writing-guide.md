# 深掘り対話ガイド

## 素材分析の6観点

AI はユーザーから投入された素材（テーマ・キーワード、断片メモ）を以下の6観点で分析し、不足している要素を特定して対話で補完する。

### 1. 具体性

素材に固有名詞・数値・技術名が含まれているか。

- 不足シグナル: 「開発を改善した」「ツールを導入した」のように漠然としている
- 深掘り方向: 具体的な技術名、ツール名、数値を聞く
- 質問例:
  - 導入した技術やツールの名前を教えてください
  - 改善の結果、数値としてどのくらい変化がありましたか
  - どのプロダクト・サービスでの話ですか

### 2. エピソード

出来事や体験の記述があるか。

- 不足シグナル: 概念や方針の説明のみで、実体験が語られていない
- 深掘り方向: 印象的なエピソードや具体的な場面を聞く
- 質問例:
  - その取り組みで一番印象に残っている場面を教えてください
  - うまくいかなかった瞬間や壁にぶつかった経験はありますか
  - 「これはやってよかった」と感じた具体的な出来事は何ですか

### 3. 背景・動機

「なぜ」「どう始まった」が明確か。

- 不足シグナル: 結果や現状の記述はあるが、始めた理由や経緯が語られていない
- 深掘り方向: きっかけ、動機、当時の課題感を聞く
- 質問例:
  - その取り組みを始めたきっかけは何でしたか
  - 当時、どんな課題を感じていましたか
  - なぜその方法を選んだのですか

### 4. 成果・変化

Before/After が見えるか。

- 不足シグナル: 取り組みの内容はあるが、結果や変化が不明
- 深掘り方向: 導入前後の比較、定量・定性的な変化を聞く
- 質問例:
  - 取り組みの前と後で、何がどう変わりましたか
  - チームやプロダクトにどんな影響がありましたか
  - 定量的に測れる変化はありますか（速度、件数、時間など）

### 5. 感情・実感

事実の羅列だけでなく温度感があるか。

- 不足シグナル: 客観的な事実のみで、感想や感情が一切ない
- 深掘り方向: やってみてどう感じたか、どんな気持ちだったかを聞く
- 質問例:
  - その経験を振り返って、どんな気持ちですか
  - 一番やりがいを感じた瞬間はいつですか
  - チームメンバーの反応はどうでしたか

### 6. メッセージ性

読者に何を伝えたいかが明確か。

- 不足シグナル: 素材はあるが「この記事で何を伝えたいか」が定まっていない
- 深掘り方向: 一番伝えたいこと、読者に持ち帰ってほしいことを言語化してもらう
- 質問例:
  - この記事を読んだ人に一番伝えたいことは何ですか
  - この話を通じて、読者にどんな印象を持ってほしいですか
  - 候補者がこの記事を読んで「入社したい」と思うポイントはどこですか

## 質問選択アルゴリズム

1. 投入された素材を6観点でスコアリングし、最も不足している観点から質問を開始する
2. 合計3〜6問を目安とする（素材の充実度に応じて調整）
3. 前の回答の内容に応じて次の質問を選定する（例: エピソードが出てきたら、そのエピソードの「感情・実感」を深掘り）
4. 最後の質問は「メッセージ性」の観点から選び、記事の核心を明確にする

## 深掘りルール

### トリガー条件

以下のいずれかに該当する場合、深掘り質問を追加する。

| 条件 | 判定基準 | 深掘りアクション |
|---|---|---|
| 短すぎる回答 | 回答が30文字未満 | 「もう少し具体的に教えていただけますか？」とエピソードを求める |
| 抽象的な表現 | 「良い」「大変」「すごい」「頑張った」等の形容詞のみ | 「具体的にどんな場面でそう感じましたか？」と場面を求める |
| 「なぜ」の欠如 | 結果のみで動機や理由が語られていない | 「そうなった背景や理由を教えてください」と経緯を求める |
| 数値の欠如 | 成果について語っているが定量データがない | 「具体的な数字はありますか？（例: 〜倍、〜%改善など）」と定量化を促す |

### 制約

- 同一質問に対する深掘りは最大2回まで
- 深掘り後も具体性が不足する場合は次の質問に移る
- 深掘り質問はプレッシャーを与えない柔らかい口調にする
- ユーザーが「わからない」「覚えていない」と回答した場合は無理に追わず次へ進む

## 行動経済学を意識した素材の引き出し

深掘り対話の中で、記事執筆時に行動経済学テクニックを活用できる素材を意識的に引き出す。以下のテクニックに対応する素材が不足している場合、対話の中で自然に聞き出す。

| テクニック | 引き出すべき素材 | 質問の方向性 |
|---|---|---|
| アンカリング | インパクトのある数値・成果 | 「具体的な数字で表すとどのくらいですか？」「改善前後で数値はどう変わりましたか？」 |
| 社会的証明 | 周囲の反応・チームの同調 | 「チームメンバーの反応はどうでしたか？」「他のチームからの反響はありましたか？」 |
| 損失回避 | 環境の希少性・機会の貴重さ | 「この経験は他の環境でもできると思いますか？」「このタイミングだからこそできたことはありますか？」 |
| フレーミング | 課題のポジティブな側面 | 「その困難を振り返って、成長につながった部分はありますか？」 |
| ピーク・エンド | 最も感情が動いた瞬間 | 「この取り組みで一番興奮した瞬間はいつですか？」「一番達成感を感じた場面を教えてください」 |
| 物語性 | 時系列の変化・転機 | 「きっかけから現在までの流れを教えてください」「転機になった出来事はありますか？」 |
| 内集団バイアス | チームの共通価値観・仲間意識 | 「チームで大切にしている価値観は何ですか？」「どんな人と一緒に働きたいですか？」 |

### 素材分析への組み込み

6観点の素材分析（具体性、エピソード、背景・動機、成果・変化、感情・実感、メッセージ性）に加えて、上記の行動経済学テクニックに対応する素材が十分に集まっているかも評価する。特に以下の素材は優先度が高い。

- **数値データ**（アンカリング用）: 定量的な成果や変化
- **他者の声**（社会的証明用）: チームメンバーや他部署からの反応
- **転機のエピソード**（ピーク・エンド用）: 最も感情が動いた瞬間
- **ストーリーの起承転結**（物語性用）: 課題→挑戦→成果の流れ
