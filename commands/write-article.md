# /write-article - 書きたいことドリブン記事作成

断片メモやテーマ・キーワードを出発点に、対話で素材を深掘りしながら採用広報・社内広報記事のドラフトを生成します。

## 実行手順

1. `.claude/skills/write-article/SKILL.md` を読み込む
2. SKILL.md に記載されたワークフロー（Phase 1〜5）を順番に実行する

## ワークフロー概要

```
Phase 1: 素材投入
  → 入力タイプ選択（テーマ / 断片メモ / 両方）
  → 素材の入力
  → 記事の種類・ターゲット・主役の情報を取得

Phase 2: 素材の深掘り対話
  → 6つの観点（具体性・エピソード・背景/動機・成果/変化・感情/実感・メッセージ性）で分析
  → 不足要素を対話で補完（3〜6問）
  → 抽象的な回答には深掘り質問を追加

Phase 3: 構成提案
  → 仮タイトル・セクション構成を提示
  → ユーザーの承認を得る

Phase 4: ドラフト生成
  → write-article-drafter サブエージェントが記事を生成
  → textlint で品質チェック・自動修正

Phase 5: レビュー・仕上げ
  → ユーザーに記事を提示し、修正の要否を確認
```

## 出力

- `articles/{YYYY-MM-DD}-{テーマのslug}.md` に記事ファイルを出力する
- textlint チェック済みの状態で納品する
