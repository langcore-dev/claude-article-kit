# /interview - 採用広報インタビュー

自社エンジニアへの対話型インタビューを実施し、採用広報記事のドラフトを生成します。

## 実行手順

1. `.claude/skills/interview/SKILL.md` を読み込む
2. SKILL.md に記載されたワークフロー（Phase 1〜4）を順番に実行する

## ワークフロー概要

```
Phase 1: コンテキスト初期化
  → 記事の目的・ターゲットペルソナ・対象者情報を取得

Phase 2: インタビュー実行
  → 5つのカテゴリから動的に質問を生成し、1問ずつ対話形式で実施
  → 抽象的な回答には深掘り質問を追加

Phase 3: ドラフト生成
  → article-writer サブエージェントが記事を生成
  → textlint で品質チェック・自動修正

Phase 4: レビュー・仕上げ
  → ユーザーに記事を提示し、修正の要否を確認
```

## 出力

- `articles/{YYYY-MM-DD}-{名前}.md` に記事ファイルを出力する
- textlint チェック済みの状態で納品する
